Secure File Protocol

Follow the step-by-step guide to encrypt and share your files anonymously.

01. Single File Upload Flow
01

Define Key Phrase

Before uploading, you must set a mandatory key phrase required for later decryption.

Requirement: Must include at least one alphabet (A-Z).
02

Upload File Packet

Select your file; the system securely links it to your defined key phrase.

03

Get Access Code

Receive a unique system-generated code that identifies your uploaded packet.

04

Share Credentials

Provide the recipient with both the unique code and the secret key phrase.

05

Decrypt & Download

The recipient enters the code and phrase to extract the file securely.

02. Multi-File Sharing Flow
01

Pre-set Master Key

Set your key phrase first. It must contain at least one alphabet to be valid for the multi-file protocol.

02

Upload Batch (Max 10)

Select multiple files. You can upload up to 10 files in a single encrypted packet.

03

Unified Access Code

The app creates one single code that represents the entire packet of 10 files.

04

Sync Access

Recipients use the single code and phrase to unlock the entire collection at once.

Protocol Highlights

Alpha-Validation: All key phrases must contain at least one alphabet character for security.

Packet Logic: The key phrase applies to the entire packet, not individual files.

Capacity: A strict 10-file limit ensures manageable and secure transmissions.
